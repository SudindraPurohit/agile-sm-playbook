# Agile SM Playbook

> A practical, opinionated collection of templates, checklists, and frameworks built from 5+ years of hands-on Scrum Master experience across eCommerce, Banking, Oracle EBS O2C, and Airlines domains.

This is not a textbook. Every file here has been used in real delivery contexts.

---

## What's inside

| Folder | Contents |
|--------|----------|
| [`/templates`](./templates/) | Retrospective formats, sprint planning agenda, stakeholder update template |
| [`/checklists`](./checklists/) | Definition of Done, Definition of Ready, PI Planning prep, impediment log |
| [`/frameworks`](./frameworks/) | Dependency tracking, impediment escalation path, team health radar |

---

## Who this is for

- Scrum Masters looking for ready-to-use templates they can adapt immediately
- Teams transitioning to Agile who need structured starting points
- Agile coaches building a facilitation toolkit

---

## How to use this

Each file is standalone. Open it, read the facilitator notes at the top, adapt to your team's context, and use it. Nothing here requires a tool or paid platform — it all works in a whiteboard, Confluence, or Miro.
